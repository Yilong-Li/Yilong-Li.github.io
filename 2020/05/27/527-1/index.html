    考期将至，这些小软件帮你提高生产力

    不知不觉间，本学期的课程也接近尾声。而伴随着结课铃，期末复习（预习）的紧密安排和各科大作业的DDL也让人倍感压力……

    今天小编就与大家分享几个能提高生产力的Windows工具。如果有其他推荐，也欢迎在评论区分享~

    有道词典

    期末大作业查资料向来令人头秃，在阅读外文资料时更是如此。而借助有道词典，可以快速实现划词与取词翻译，在浏览外文网页、阅读代码时也很有作用。

    网易有道词典支持中文、英语、日语、韩语、法语、德语、俄语、西班牙语、葡萄牙语等108种语言翻译。配合快捷键，可以选中文字并进行翻译。

    Quicker

    现在鼠标除了经典的鼠标左键、右键、滚轮中键，往往还会有前进、后退的侧键。而Quicker就是一款充分利用鼠标按键的快捷功能工具。

    Quicker可以根据你正在使用的软件自动加载预设的动作，进行文本OCR、截图、剪贴板翻译等操作。

    Snipaste

    Snipaste是一款安全无广告的免费软件，即「Snip截图+Paste贴图」。软件的核心用法很简单，下载并打开 Snipaste，按下 F1 来开始截图，再按 F3，截图就在桌面置顶显示了。

    此外，你还可以将剪贴板里的文字或者颜色信息转化为图片窗口，并且将它们进行缩放、旋转、翻转、设为半透明、标记等，便于进行分享。

    Everything

    Everything是 Windows 上一款搜索引擎，能够基于文件名快速定文件和文件夹位置。与 Windows 内置搜索相比，”Everything” 默认显示电脑上每个文件和文件夹 (就如其名 “Everything”)。在搜索框输入的关键词将会筛选显示的文件和文件夹。

    Everything可以帮助你查找平时的课程资料，对于NTFS格式的硬盘，只需要1秒就能建立约120000文件的索引，借助正则表达式等搜索语法，可以快速找到你需要的文件。

    QuickLook

    QuickLook 是一款开源、免费的桌面快速预览工具。只需要用鼠标选中文件，然后点击空格，即可预览。就像 macOS 里的那样，支持图片、压缩包、PDF、Office 文档、视频等等。

    搭配Everything可以快速预览搜索结果，便于整理期末资料。除了上述的文件格式，还可以通过安装插件进一步优化预览的体验。

    以上就是今天分享的几个小工具，祝大家都能在学期末取得好成绩~
